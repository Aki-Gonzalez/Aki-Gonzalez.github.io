Professional Work The Game Assembly


IronWrought is an engine built by the programmers of the group SoftBlob Studios during the second year at The Game Assembly. The engine is written in C++, uses DirectX11 as it graphics API and runs PhysX for its physics simulation. IronWrought used Unity 3D as its level editor.

Core parts of the engine that affected gameplay were developed by several programmers to ensure that we always had someone available that was responsible and had a good understanding of a system. I shared responsibility over the Object and Component system as well as the State Stack.

Other contributions to the engine were: a line drawer for visual debugging and an animation state machine.
